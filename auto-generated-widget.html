    Board Layer(s)

    Select which board layer(s) to render. Toolpaths and Gcode will be generated for the selected layer(s) only.

    Flip Axis

    Select which board items to see in the 3D viewer, these selection has no effect on toolpaths and Gcode generation.

    This allows you to hide the popup windows as you mouse around with the 3D viewer. They can sometimes get in the way when iterating the best endmill sizes and inflate by values.
